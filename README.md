# To-Do List in Rust

A simple to-do list application built with Rust and the Rocket framework.

## Features

*   Fetches tasks from a `tasks.csv` file.
*   Exposes a `/tasks` endpoint to view the tasks.

## Getting Started

1.  Install Rust: [https://www.rust-lang.org/tools/install](https://www.rust-lang.org/tools/install)
2.  Clone the repository: `git clone https://github.com/Sunil-Reddy-Gummalla/todo_rust.git`
3.  Navigate to the project directory: `cd todo_rust`
4.  Run the application: `cargo run`
5.  Access the tasks at `http://127.0.0.1:8000/tasks`
